1.	封皮统一按照 [数据库课程实验-封面.doc](数据库课程实验-封面.doc) 格式，除封面外其它内容一律双面打印。

  2. 数据库的完整性和安全性

6.	报告统一在考试当日上交，过期按缺考处理。
